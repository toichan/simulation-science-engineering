# 問題3 レポート用メモ

## 課題設定

- 問題1: 2次元に制限した条件
- 問題2: 不安定軌道の計算（自発イオン化）
- 目的: 固定時間幅法と適応的時間幅法を比較し、適応法の有効性を調べる

## 計算原理

- 原点に固定した原子核と2個の電子からなる古典的ヘリウム原子を扱う
- 電子1, 2 の加速度は次式で与える

  a1 = -2 r1 / |r1|^3 + (r1 - r2) / |r1 - r2|^3

  a2 = -2 r2 / |r2|^3 + (r2 - r1) / |r2 - r1|^3

- 全エネルギーは運動エネルギーとポテンシャルエネルギーの和

  E = 1/2 (|v1|^2 + |v2|^2) - 2/|r1| - 2/|r2| + 1/|r1 - r2|

## プログラムの考え方

- C99で実装する
- 状態は配列で持ち、座標成分は enum で管理する
- 固定時間幅法と適応的時間幅法を同じ時間発展関数で比較する
- 適応法では 1回の dt と 2回の dt/2 の結果を比較し、全エネルギー差から dt を調整する

## 考察の観点

- エネルギー保存誤差がどの程度抑えられたか
- 問題2のような不安定軌道で dt がどのように変化したか
- 固定時間幅より少ない計算量で同等以上の精度が得られたか
- 近接遭遇時に適応法が有効に働いたか
